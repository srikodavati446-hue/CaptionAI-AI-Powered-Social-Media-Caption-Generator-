# CaptionAI-AI-Powered-Social-Media-Caption-Generator-
CaptionAI is an AI-powered platform that generates platform-optimized social media content. Built with Flask and driven by Groq’s API via the LLaMA 3.3-70B model, it delivers three tailored captions, ten curated hashtags, and three punchy CTAs per request. Includes secure env management and public deployment via Ngrok.
